## Dependencies

You will need to have both `bluestacks5` and `pip` installed on your system and then install the following using pip
```
$ pip install cv2
$ pip install PIL
$ pip install numpy
$ pip install pynput
$ pip install pyautogui
$ pip install pyperclip
$ pip install pytesseract
```
